SVG Logo Design

This assignment involved creating a scalable vector graphic (SVG) logo with knocked-out initials that links back to my assignments directory.

My SVG Logo

Logo with "ED" initials knocked out of a circular shape

Implementation Details

  • Technique: SVG mask for knockout effect
  • Elements: Circle shape with text mask
  • Accessibility: Title element for screen readers
  • Interactivity: CSS hover and active states
  • Scalability: Vector-based for infinite resolution

CSS Styling


.svg-logo circle {
  fill: #2c3e50;
  transition: fill 0.3s ease;
}

.svg-logo:hover circle {
  fill: #e67e22;
}

.svg-logo:active circle {
  fill: #c0392b;
}
